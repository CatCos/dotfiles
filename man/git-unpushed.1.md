# NAME

git-unpushed - Show unpushed work

# SYNOPSIS

*git unpushed* [\<options\>]

# DESCRIPTION

Shows the unpushed work.

If no options are given, unpushed commit logs are shown.

# OPTIONS

-d, \--diff
: View the changes made relative to HEAD
